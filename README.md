# Парсер каталога Wildberries

Парсер товаров с Wildberries по поисковому запросу с сохранением в XLSX формате.

## Возможности

- Поиск товаров по запросу
- Получение детальной информации о каждом товаре
- Сохранение полного каталога в XLSX
- Фильтрация по рейтингу, цене и стране производства
- Rate limiting и retry логика для стабильности
- Красивое форматирование Excel файлов

## Установка
```bash
# Клонировать репозиторий
git clone <repository-url>
cd wildberries-parser

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установить зависимости
pip install -r requirements.txt
```

## Использование
```bash
python main.py
```

Парсер:
1. Ищет товары по запросу "пальто из натуральной шерсти"
2. Собирает детальную информацию о каждом товаре
3. Сохраняет полный каталог в `output/full_catalog.xlsx`
4. Фильтрует товары (рейтинг ≥4.5, цена ≤10000₽, Россия)
5. Сохраняет отфильтрованный каталог в `output/filtered_catalog.xlsx`

## Собираемые данные

- Ссылка на товар
- Артикул
- Название
- Цена
- Описание
- Ссылки на изображения
- Характеристики товара
- Название и ссылка на селлера
- Размеры
- Остатки
- Рейтинг
- Количество отзывов

## Настройка

Параметры можно изменить в файле `src/config.py`:
```python
# Поисковый запрос
SEARCH_QUERY = "ваш запрос"

# Параметры фильтрации
FILTER_RATING_MIN = 4.5
FILTER_PRICE_MAX = 10000
FILTER_COUNTRY = "Россия"

# Rate limiting
REQUEST_DELAY = 1.0  # секунд между запросами
```

## Особенности реализации

- **Rate limiting** — задержка между запросами чтобы не попасть в бан
- **Retry логика** — автоматические повторные попытки при ошибках
- **User-Agent rotation** — случайные User-Agent для имитации браузера
- **Batch requests** — запрос информации пачками до 100 товаров
- **Error handling** — корректная обработка ошибок и продолжение работы
